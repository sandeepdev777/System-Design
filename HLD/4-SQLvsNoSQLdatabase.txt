| Feature                | SQL Databases                | NoSQL Databases                     |
|------------------------|----------------------------- |-----------------------------        |
| Data Model             | Relational (tables)          | Document, Key-Value, Graph, Column  |
| Schema                 | Fixed, predefined            | Dynamic, flexible                   |
| Scalability            | Vertical scaling             | Horizontal scaling                  |
| Transactions           | ACID compliant               | BASE, eventual consistency          |
| Query Language         | SQL                          | Varies (e.g., JSON, CQL)            |
| Examples               | MySQL, PostgreSQL, Oracle    | MongoDB, Cassandra, Redis           |
| Best Use Case          | When consistency of data is required | Big data, rapid development |
